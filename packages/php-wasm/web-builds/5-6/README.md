# @php-wasm/web-5-6

PHP 5.6 WebAssembly binaries for the web.

This package provides PHP 5.6 compiled to WebAssembly for use in web browser environments. It is part of the WordPress Playground project.

## Usage

```js
import { getPHPLoaderModule } from '@php-wasm/web-5-6';

const phpLoaderModule = await getPHPLoaderModule();
```

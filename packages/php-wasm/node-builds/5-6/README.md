# @php-wasm/node-5-6

PHP 5.6 WebAssembly binaries for Node.js.

This package provides PHP 5.6 compiled to WebAssembly for use in Node.js environments. It is part of the WordPress Playground project.

## Usage

```js
import { getPHPLoaderModule } from '@php-wasm/node-5-6';

const phpLoaderModule = await getPHPLoaderModule();
```
